# DiceDuel
A simple Python dice game where the player competes against the computer by choosing when to roll or hold. The first to reach 100 points wins.

## Features
- Turn-based dice game between player and computer  
- Roll or hold decision system with risk-based scoring  
- Computer opponent with human-like decision making  
- Score tracking across rounds  
- First-to-100 win condition  
- Simple text-based interface  

## Requirements
- Python 3.8 or higher
